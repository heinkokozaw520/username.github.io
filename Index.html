🗺️ Master Roadmap

1
Phase 1 — English + GED Highest Priority ~12–18 months

Reach Intermediate English level and pass all four GED subject tests to qualify for university enrollment at University of the People (UoPeople).

🇬🇧 English Grammar 📖 Vocabulary 👂 Listening 🗣️ Speaking 📚 Reading ✏️ Writing 🔢 GED Math 🔬 GED Science 🌍 GED Social Studies 📝 GED Writing

Pass all 4 GED exams + achieve B1 English level → Apply to UoPeople Computer Science

2
Phase 2 — Web Programming After Phase 1 ~9–15 months

Build a strong foundation in web development by learning HTML, CSS, and JavaScript, then automate workflows with Google Apps Script. Complete multiple real-world projects before advancing.

🌐 HTML 🎨 CSS ⚡ JavaScript 🤖 Google Apps Script 🛠️ Projects ×10

Complete 10 projects covering HTML → CSS → JS → GAS before starting Phase 3

3
Phase 3 — Chinese (HSK 1–5) Do NOT start before Phase 2 ~24–36 months

Learn Mandarin Chinese systematically from beginner (HSK1) through upper-intermediate (HSK5), building vocabulary, reading, listening, and speaking across all levels.

🈶 HSK 1 — Beginner 🈶 HSK 2 — Elementary 🈶 HSK 3 — Pre-Intermediate 🈶 HSK 4 — Intermediate 🈶 HSK 5 — Upper-Intermediate

Pass official HSK 5 exam with a score of 180+ points
